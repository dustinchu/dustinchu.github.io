## BOM查詢系統小程式 ##

### 前言 ###
> 版權聲明：轉載請保留原文連結及作者
這個小程式主要用途在於
讓資材key上料號把bom展出來 有替代料的點下右鍵
會在下方出現替代料
比較麻煩的部分是 一階BOM展開二階時候二階*上1階的數量這樣才對


### 程式主畫面 ###





# works # java # swing # tiptop
java-swing tiptop出貨排程小程式
java swing tiptop異動數量查詢
